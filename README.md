##java-basics

A collection of beginner-level Java programs created while learning fundamental programming concepts.

## Contents

* **BMICalculator.java** – Calculates Body Mass Index (BMI) based on user input.
* **ChineseZodiac.java** – Determines the Chinese Zodiac sign for a given year.
* **CircleAreaComputer.java** – Calculates the area of a circle.
* **HexadecimalToDecimal.java** – Converts a hexadecimal digit to its decimal equivalent.
* **Lottery.java** – A simple lottery game using random number generation.
* **PerpendicularBisectorCalculator.java** – Calculates the equation of the perpendicular bisector of two points.
* **ThreeNumberSorter.java** – Sorts three numbers in descending order.

## Topics Covered

* Variables and data types
* User input with Scanner
* Arithmetic operations
* Conditional statements (if-else, switch)
* Loops
* String manipulation
* Methods
* Basic mathematical calculations
